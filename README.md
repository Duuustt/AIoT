# AIoT-
架构设计：自主搭建基于 AIoT 架构 的实时监控系统，采用 MQTT 协议 实现了边缘端设备与云服务器之间的轻量级异步数据传输，解决了高并发场景下的视频流传输延迟问题。  算法实现：负责核心检测模块，部署 YOLOv8 深度学习模型实现对人、车等多种目标的精准识别；通过对比测试不同规模模型（n/s/m/l），在云端有限算力下优化了检测帧率（FPS）。  后端开发：使用 FastAPI 框架构建高性能异步后端接口，实现了检测结果与 Web 前端的可视化联动，确保了监控反馈的实时性。  云端运维：独立完成项目在 阿里云 Linux 服务器 的生产环境部署，负责安全组配置、端口管理及网络调优，确保了系统 7×24 小时的稳定运行。
